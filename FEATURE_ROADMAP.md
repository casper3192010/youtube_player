## 🚧 進行中功能

### WebView 播放清單讀取 (Phase 1)
**狀態:** 開發中  
**優先級:** 高  
**預計完成:** 2026-01-21

#### 功能描述
利用 WebView 和 JavaScript 注入，無需複雜 API 配置，直接讀取 YouTube 播放清單。

#### 實現範圍
1. ✅ 稍後觀看列表 (WL)
2. ⏳ 喜歡的影片 (LL)
3. ⏳ 自定義播放清單

#### 技術方案
- **方式:** WebView + JavaScript 注入
- **優勢:** 
  - 無需 API Key
  - 無配額限制
  - 與現有架構一致
  - 用戶熟悉的登入流程
