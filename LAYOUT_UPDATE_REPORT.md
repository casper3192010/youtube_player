# 控制按鈕佈局更新報告

## 📅 更新日期
2026-01-21

## 🎨 新佈局設計

### 第一排:主要播放控制
```
[加入收藏] [我的收藏] [歷史紀錄] [倒退15秒] [▶️ 播放/暫停] [快進30秒]
```

**按鈕順序(從左到右):**
1. **加入收藏** (btn_action_save) - 添加到收藏分類
2. **我的收藏** (btn_favorite) - 查看收藏列表
3. **歷史紀錄** (btn_history) - 查看播放歷史
4. **倒退 15 秒** (btn_rewind) - 快速倒退
5. **播放/暫停** (btn_play_pause) - 主控制按鈕(較大)
6. **快進 30 秒** (btn_forward) - 快速前進

### 第二排:速度控制 + PIP
```
[PIP]  播放速度： [-] [1.000x] [+]
```

**按鈕順序(從左到右):**
1. **PIP 按鈕** (btn_pip_small) - 畫中畫模式
2. **播放速度標籤** - 文字說明
3. **減速** (btn_speed_down) - 降低播放速度
4. **速度顯示** (tv_speed_display) - 當前速度
5. **加速** (btn_speed_up) - 提高播放速度

## 💡 設計理念

### 功能分組
1. **收藏管理區** (左側)
   - 加入收藏 → 我的收藏 → 歷史紀錄
   - 邏輯流程:添加 → 查看 → 歷史

2. **播放控制區** (中央)
   - 倒退 → 播放/暫停 → 快進
   - 核心播放功能

3. **進階功能區** (第二排)
   - PIP + 速度調整
   - 次要但實用的功能

### 用戶體驗優化
- ✅ **相關功能靠近**: 收藏和歷史功能放在一起
- ✅ **視覺平衡**: 第一排不再擁擠
- ✅ **操作流暢**: 常用功能在第一排,進階功能在第二排
- ✅ **邏輯清晰**: 從左到右的操作流程更自然

## 📐 按鈕尺寸

## 🎯 按鈕功能說明

### 收藏管理組
| 按鈕 | 圖標 | 功能 | 操作 |
|------|------|------|------|
| 加入收藏 | 播放列表+ | 添加當前影片到收藏 | 單擊:選擇分類 |
| 我的收藏 | 星星 | 查看收藏列表 | 單擊:查看<br>長按:快速收藏 |
| 歷史紀錄 | 時鐘 | 查看播放歷史 | 單擊:查看歷史 |

| 按鈕 | 圖標 | 功能 |
|------|------|------|
| 倒退 15 秒 | ⟲10 | 快速倒退 15 秒 |
| 播放/暫停 | ▶️/⏸ | 控制播放狀態 |
| 快進 30 秒 | ⟳30 | 快速前進 30 秒 |

| 按鈕 | 圖標 | 功能 |
|------|------|------|
